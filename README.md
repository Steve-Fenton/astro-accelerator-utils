## Update Type Information

Run the command:

    pnpm types

## Running Tests

Run the command:

    pnpm test

## Create new version

Run the command:

    pnpm refresh

This updates dependencies, increments the version number, and re-runs types and tests. If successful, committing the change updates the package on NPM.
